# Cold Message Generator

An AI-powered tool to generate personalized cold outreach messages for LinkedIn, Email, and more.

## Features

- **LinkedIn Scraping**: Extracts profile data (experience, education, recent posts).
- **AI Analysis**: Uses LLM to analyze profiles and generate psychological insights.
- **Multi-Channel Generation**: Creates tailored messages for Email, LinkedIn, WhatsApp, and SMS.
- **Knowledge Base**: Stores prospects and uses "Social Proof" to reference past successful connections.
- **Offering-Aware**: Adapts social proof based on whether you are selling a Bootcamp, Talent/Hiring, or Dev Tools.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   streamlit run app.py
   ```
